# My Blog

A personal blog built with Astro, based on the Fuwari theme and customized for my own needs.

## About

This is the source code for my personal blog.
The project is based on the Fuwari theme and includes custom modifications to styling, layout, and content structure.

## Tech Stack

- Astro
- Tailwind CSS
- TypeScript
- Markdown / MDX
